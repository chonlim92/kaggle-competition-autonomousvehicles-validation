--- ACTIVE KNOWN ISSUE ---

BUG ID          : BUG-CAM-402-WET-007
Status          : ACTIVE — UNRESOLVED
Severity        : HIGH
Introduced In   : v4.2.1-beta (PATCH-402-V421B)
Discovered      : 2026-05-11T07:18:00Z (post-deployment field observation)
Reported By     : Fleet Ops, Zone 7 — Shift Lead
Assigned To     : Perception Engineering Team — Camera Pipeline
ETA Fix         : v4.2.2 (scheduled 2026-06-28)
Workaround      : Active (see AV-REG-102 Wet-Road Camera Reflection Protocol)

DESCRIPTION:
  AV-FLEET-402 exhibits a camera reflection processing anomaly exclusively on
  wet road surfaces. When road_surface=WET is detected by the environmental
  context module, the forward camera array produces specular reflection artefacts
  that are incorrectly classified as phantom objects by the perception module's
  v4.2.1-beta object detection pipeline.

  Root cause (preliminary): The beta patch modified the image pre-processing
  normalisation layer to improve low-light intersection detection. An unintended
  side effect is that the normalisation curve amplifies specular highlight regions
  on wet asphalt, causing the downstream YOLO-based detector to generate false
  positive bounding boxes for non-existent objects (primarily at road level,
  typically classified as 'movable_object.barrier' or 'vehicle.car').

  Impact profile:
    - Condition trigger : road_surface=WET; precipitation_level ≥ LIGHT
    - Affected sensors  : Forward camera array (cam_front_left, cam_front_right,
                          cam_front_centre). Surround cameras NOT affected.
    - Phantom object rate: ~12–18 false positive objects per wet-road scene
    - Confidence range  : False positives appear with confidence 0.62–0.74
                          (below the 0.75 human-reviewed threshold, but above
                          the 0.50 autonomous action threshold — SAFETY CRITICAL)
    - AV-REG-101 impact : Phantom objects near intersections can trigger spurious
                          emergency decelerations, causing AV-REG-101 compliance
                          records to show false velocity anomalies
    - AV-REG-102 impact : Does NOT trigger sensor obstruction alerts (sensor hardware
                          is functioning; the bug is in software processing)

  Validation agent note:
    When analysing telemetry from AV-FLEET-402 where road_surface=WET:
      1. Apply AV-REG-102 confidence penalty multiplier (1.5×) to all cam_front_*
         detections before evaluating against detection thresholds.
      2. Flag any 'movable_object.barrier' or 'vehicle.car' detection with
         confidence 0.62–0.74 as SUSPECT_PHANTOM — do not count as true positive.
      3. Do NOT raise AV-REG-102 Level 2/3 alerts solely on the basis of phantom
         objects from this bug — cross-validate with LiDAR before escalating.
      4. Flag the scene with tag: BUG-CAM-402-WET-007 for engineering review.
